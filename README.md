# 🐍 Python File Organizer
**A clean-up tool for busy desktops.**

## 📖 What it does
This script automatically scans my Desktop and moves images (PNG/JPG) and PDFs into organized folders. It is designed to save time and keep my workspace professional.

## 🛡️ Safety Features
- **Sensitive File Protection:** The script will automatically skip any file that has the word "IMPORTANT" in the name.
- **Dry Run Logic:** It prints a message in the Terminal before moving anything so you know exactly what is happening.

## 🚀 How to Use It
1. Open your **Terminal**.
2. Navigate to your home folder: `cd ~`
3. Run the script: `python3 organizer.py`

## 🛠️ Built With
* **Python 3** - The core programming language.
* **OS & Shutil Libraries** - Used for interacting with the Mac file system.
* **VS Code** - My primary code editor.
